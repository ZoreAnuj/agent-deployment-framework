# Agent Deployment Framework

A streamlined framework for rapidly deploying AI agents into production applications. It enables developers to build, test, and scale agentic workflows, with the option to deploy via a managed platform or self-host for full control. This project explores efficient patterns for integrating autonomous agents into real-world systems.

## Key Features
*   Rapid prototyping and deployment of multi-agent workflows.
*   Support for both cloud (MagmaDeploy) and self-hosted deployment options.
*   Built-in tools for agent communication, tool calling, and state management.

## Tech Stack
Python, FastAPI, LangChain, Pydantic, Docker

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/agent-deployment-framework.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Configure your environment variables and run the example agent.
